# Performance Monitoring Agent Skills

## Skill 1: Response Time Measurement

**Purpose:** Measure endpoint latency and response times
**Tools:** requests, timing
**Complexity:** Intermediate

### Procedure

1. Send requests to all endpoints
2. Measure response time for each
3. Calculate average and percentiles
4. Identify slow endpoints
5. Generate performance report

### Validation Checklist

- ✓ Latency measured
- ✓ Baselines established
- ✓ Percentiles calculated
- ✓ Slow endpoints identified
- ✓ Trends tracked

# BIST Verification Agent Skills

## Skill 1: BIST Test Execution

**Purpose:** Execute all BIST tests
**Tools:** pytest, bist_runner
**Complexity:** Intermediate

### Procedure

1. Discover all BIST tests
2. Execute test suite
3. Monitor test execution
4. Collect test metrics
5. Track execution time

### Validation Checklist

- ✓ All tests discovered
- ✓ Tests executed
- ✓ Results collected
- ✓ Execution time recorded
- ✓ Status reported

## Skill 2: Result Analysis

**Purpose:** Analyze BIST results
**Tools:** Result parser
**Complexity:** Intermediate

### Procedure

1. Parse test results
2. Compare with baseline
3. Identify passing tests
4. Identify failing tests
5. Track success rate

### Validation Checklist

- ✓ Results parsed
- ✓ Baseline compared
- ✓ Pass/fail counted
- ✓ Success rate calculated
- ✓ Trends identified

## Skill 3: Failure Detection

**Purpose:** Detect and analyze test failures
**Tools:** Result parser
**Complexity:** Intermediate

### Procedure

1. Identify failed tests
2. Extract error messages
3. Categorize failure types
4. Identify patterns
5. Report root causes

### Validation Checklist

- ✓ Failures identified
- ✓ Error messages captured
- ✓ Stack traces included
- ✓ Patterns detected
- ✓ Issues reported
